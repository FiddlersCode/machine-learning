## Run locally
View the repository [here](https://github.com/pmuldoon86/machine-learning) on GitHub

Clone the repo. Then:

```bash
bundle install
bin/rails db:create
bin/rails db:migrate
```
#### To run the tests
```bash
bundle exec rspec
```

### To view in browser
```bash
bin/rails server
```
Then start the server at localhost:3000
